# My Friend's Paint 3D Model

This is a 3D model originally created by my friend using **Microsoft Paint 3D**. I made a quick recreation of it and open-sourced it for fun.

---

## 📝 How to Use
1. Download the model file from this repository.
2. Open it directly in **Microsoft Paint 3D** (Windows built-in app).
3. You can view, edit, or remix it however you like.
